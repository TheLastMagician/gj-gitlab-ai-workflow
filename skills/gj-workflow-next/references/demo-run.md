# Demo Run Reference

First-run stage trace:

1. Bootstrap assets.
2. Requirement refine.
3. Solution plan.
4. Issue split.
5. Dev context and implementation.
6. MR review.
7. Test design and Bug Issue.
8. Release prep.
9. Retro learnings.
10. Context extract.

When asked "what next", inspect evidence in this order:

1. Git status.
2. Open MR and latest pipeline.
3. Current Issue labels and comments.
4. Missing required artifacts.
5. Human confirmation points.
